# Scribble: Command Line Word Game

Welcome to Scribble, a Python-based command line word game that offers an interactive word-play experience! This game is designed to provide you with hours of entertainment as you challenge yourself to come up with words based on a dynamic word bank fetched from an API.

## Features

- **Interactive Gameplay**: Enjoy a captivating word game experience directly from your command line interface.
- **Dynamic Word Selection**: Scribble integrates an API to fetch English letter words, ensuring a diverse and challenging word selection every time you play.
- **User-Friendly Interface**: Designed with simplicity in mind, Scribble offers a seamless gameplay experience within the command line environment.

## How to Play

1. Clone the repository to your local machine.
2. Make sure you have Python installed.
3. Run the Scribble game script.
4. Follow the on-screen instructions to play the game.
